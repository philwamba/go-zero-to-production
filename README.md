# Go Zero to Production

Welcome to the **Go Zero to Production** learning repository. This project is structured to take you from the absolute basics of the Go programming language to building production-ready applications.

## Project Structure

The repository is organized into modules, each focusing on a specific stage of learning or development.

### [01-Basics](./01-basics/)

Foundational concepts of Go.

- [Variables](./01-basics/variables)
- [Data Types](./01-basics/data-types)
- [Conditionals](./01-basics/conditionals)
- [Loops](./01-basics/loops)

### [02-Functions](./02-functions/)

Deep dive into functions in Go.

- [Functions](./02-functions/functions)
- [Multiple Returns](./02-functions/multiple-returns)
- [Defer, Panic, Recover](./02-functions/defer-panic-recover)

## Getting Started

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/philwamba/go-zero-to-production.git
    cd go-zero-to-production
    ```

2.  **Navigate to a module:**

    ```bash
    cd 01-basics/variables
    ```

3.  **Run the code:**
    ```bash
    go run main.go
    ```

## Prerequisites

- [Go](https://go.dev/dl/) installed on your machine (version 1.21+ recommended).
